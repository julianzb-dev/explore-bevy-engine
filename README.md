# Explore Bevy Engine

A small Bevy 2D prototype for experimenting with sprite animation, simple map blocks, and Avian 2D physics.

## Run

```sh
cargo run
```

## Controls

- Move left/right: `A`/`D` or arrow keys
- Face up/down: `W`/`S` or arrow keys
- Jump: `Space`

## Notes

- Animation frames are loaded from Aseprite files in `assets/`.
- Physics, collision, gravity, and grounded checks are handled by `avian2d`.
